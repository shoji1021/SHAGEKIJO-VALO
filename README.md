# SHAGEKIJO

VALORANTの射撃場（The Range）をブラウザ上で再現した練習ツールです。
Three.jsを使用し、FPS視点でのエイム練習をブラウザで完結させることを目的としています。

- **Preview:** https://shagekijo.pages.dev/
- **Repository:** https://github.com/shoji1021/SHAGEKIJO-VALO.git

---

## 主要機能

### 1. ボットのリスポーンロジック
- フィールド上に常に10体のボットが維持されます。
- 1体を倒すと、即座に別のランダムな位置（プレイヤー前方の扇状範囲）へリスポーンします。
- 全てのボットの頭の高さ（ヘッドライン）は一定に保たれています。

### 2. ヒット判定とダメージ計算
- 頭部と胴体の部位判定を実装しています。
- **Headshot:** 160ダメージ（即死）
- **Body:** 40ダメージ
- 設定メニューからアーマーのON/OFFを切り替えることで、ボットのHPを100または150に変更可能です。

### 3. 設定カスタマイズ
- **マウス感度:** 0.01単位での詳細な調整が可能です。
- **クロスヘア:** ドット、十字、丸の3タイプから選択できます。
- **サウンド:** 射撃時にVandalを模した銃声が再生されます。連射時の音の重なりにも対応しています。

---

## 操作方法

| キー / マウス | 動作 |
| :--- | :--- |
| **左クリック** | 射撃 / メニューからの復帰 |
| **W, A, S, D** | プレイヤーの移動 |
| **マウス移動** | 視点移動 |
| **ESC** | 設定メニューの表示 / カーソルロック解除 |

---

## ローカルでの実行方法

このプロジェクトはブラウザの標準機能とThree.jsを使用しています。

1. リポジトリをクローンします。
2. `index.html`と同じディレクトリに、銃声ファイル `Vandal Shot 1.mp3` を配置してください。
3. ブラウザのセキュリティ制限を避けるため、ローカルサーバー経由での実行を推奨します。
   - VSCodeの拡張機能「Live Server」などを使用して起動してください。
